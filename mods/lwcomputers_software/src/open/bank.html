bank

The bank application provides the Mine Bank program and online transaction server.

Requires WIN

The bank computer must be labelled for online services. This label is used as the bankname by the network connection. The bank's network can be customized by including an ini file of the same name in the same folder, appended with .ini, containing the following settings.

banknetwork The bank's network (connection application id), defaults to mine_bank_network.
