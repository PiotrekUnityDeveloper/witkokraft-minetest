advertise [displayTime]

The advertise application displays a full frame html page, randomly selected from a directory at a regular interval of time.

Requires WIN

A directory named "adverts" must exist in the same directory as the advertise application. All the files in this directory are randomly selected from to display as an advert. Generally the advertise application is run on a monitor, and the advert pages should be formatted to display at the monitors resolution without extending beyond the displayable area.

displayTime The seconds for each page to display. If not given defaults to 10 seconds.
