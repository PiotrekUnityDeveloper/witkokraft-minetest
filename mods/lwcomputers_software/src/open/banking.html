banking

The banking application provides eftpos services and allows you to manage your Mine Bank account online.

Requires WIN

The bank's network can be customized by including an ini file of the same name in the same folder, appended with .ini, containing the following settings.

banknetwork The bank's network (connection application id), defaults to mine_bank_network.

bankname The bank computer's label, defaults to mineBank.

account Your bank account name. If given, forms will be prefilled.

retail Your bank account name. If given, transaction forms' to account will be prefilled.
