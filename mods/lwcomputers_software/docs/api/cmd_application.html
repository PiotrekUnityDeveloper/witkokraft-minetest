Command - WIN

Command

File: /apps/cmd

Terminal application.



Menu
Quit App Ctrl+Alt+X Quits the application, aborting any running process.

Remarks
A file path can optionally be given as an argument on the command line of the file to run. If a first argument of -i is given the Command application continues after the host program has completed. If not the Command application quits.


See also Default applications, WIN.
