Chat - WIN

Chat

File: /apps/chat

A chat application.



Menu
Clear Msgs Ctrl+M Clears the messages recorded in the center text window.
Quit App Ctrl+Alt+X Quits the application.

Remarks
Participants are identified by their computer label.

Send a message by entering it into the Message field and hitting enter, or clicking the send , ">", button.

If the Recipient field has an entry, messages are sent to that participant exclusively, otherwise they are broadcasted. If the Only check box is also checked, messages are received from that participant only.

If a message was sent exclusively the participant's name is enclosed in square brackets in the recorded messages window.

This application requires a comm connection in the system's /win.ini file. The settings of this connection must be consistent between participants.
