Manager - WIN

Manager

File: /apps/manage

Application to manage desktop and system settings.



Actions

Desktop
@ Opens a list of valid desktops to select from.
Add Adds an entry to the applications list.
Remove Removes the selected entry from the applications list.
Edit Edits the selected entry in the applications list. Also by double clicking the entry.
Up Moves the selected application entry up one position in the list.
Down Moves the selected application entry down one position in the list.
Theme Opens a dialog to edit the desktop's theme.
Startup Opens the desktop's startup.ini file in notepad.

System
System Opens the system's /win.ini file in notepad.
Name Opens a dialog to change the computer's label.
Log Opens the system's log, at /win.log, in notepad.
Clean Clears the system's log file and the temporary file folder at /win/tmp.

Remarks
The desktop actions affect the currently selected desktop only. The system actions affect the whole system.

The computer's id displays in the title bar after the colon.

If the desktop's startup.ini or system's /win.ini file is opened but does not exist, the file as when it was installed is created first.
