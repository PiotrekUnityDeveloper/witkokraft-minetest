WIN

WIN is a simple multitask desktop operating system for lwcomputers. The desktop interface is a simple, tablet like format. A single line task bar at the bottom with two buttons on the left and a button on the right. The two buttons on the left display a page each: The Home page, which contains a list of installed programs; and the List page, which displays a list of currently running programs. The button on the right displays the current date and time.

The system supports multiple simultaneous desktops; the terminal and one for each monitor that may be attached. Each desktop is independent. Each desktop object has a built in on screen keyboard which can be displayed for key input on monitors.

While the user interface is relatively simple, the programming interface is quite extensive. Allowing a full window programming interface; placing controls on a form and handling the events. Minimizing the UI and system wide programming.
