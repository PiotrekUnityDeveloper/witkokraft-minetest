Special Keys - WIN

Desktop
Ctrl+Alt+K Lock screen (if password set and nolock is false)
Alt+H Activate Home page
Alt+L Activate List page
F10 Toggle full screen

Frames and Popups
Tab Next control
Shift+Tab Prior control
Ctrl+Alt+X Quit/close the top application frame or popup window

Clipboard
Ctrl+Alt+V
 Fires a clipboard event if a clipboard is in one of the computer's slots.


See also WIN.
